## üéØ Code Generation Prompts

### 1. Perfect Code Structure
```
You are a senior software architect. Write production-ready [LANGUAGE] code for [TASK].

Requirements:
- Follow [FRAMEWORK] best practices
- Include error handling
- Add comprehensive comments
- Make it modular and testable

### 2. Code Review & Optimization
```
Review this code and suggest improvements:
[PASTE CODE]

Focus on:
1. Performance bottlenecks
2. Security vulnerabilities
3. Code maintainability
4. Best practice violations

Provide specific line-by-line suggestions.
```

### 3. Debug Like a Pro
```
I'm getting this error: [ERROR MESSAGE]

Here's the relevant code:
[CODE SNIPPET]

Debug this by:
1. Explaining what's causing the error
2. Showing the fix with code
3. Explaining why the fix works
```

## üí° Pro Tips

**1. Be Specific**
- ‚ùå "Write code for login"
- ‚úÖ "Write a secure JWT-based authentication system in Node.js with bcrypt password hashing and refresh token rotation"

**2. Provide Context**
- Always include relevant background
- Share constraints (time, budget, tech stack)
- Mention what you've already tried

**3. Iterate**
- Start broad, then refine
- Ask for alternatives
- Request explanations

**4. Use Delimiters**
```
Use ### for sections
Use ``` for code blocks
Use --- for separators
```

**5. Set Output Format**
- "Respond in JSON format"
- "Create a markdown table"
- "Use bullet points only"
